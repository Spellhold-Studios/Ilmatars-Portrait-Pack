Introduction

Hey, do you remember Ilmatar? I'm sure you do! She used to be one of the most recognized members of Spellhold Studios. She may be gone somewhere now, but her work can never die!

Ilmatar's Portrait Pack is the mod that replace 9 BG2 NPC's portraits with her made from very scratch images. So, we have here:

  • Aerie (2 versions)
  • Cernd
  • Edwin (2 versions)
  • Imoen
  • Jaheira
  • Mazzy
  • Nalia
  • Viconia
  • Yoshimo

Each portrait is installed individually, so you get to choose which portraits you use.

Visit the website or forum for all the latest updates to this mod.

Installation

PC

Ilmatar's Portrait Pack is a WeiDU mod, packaged in an extractable .RAR archive. Simply unrar "IlmatarPack v.XY.rar" (WinRAR) into your Baldur's Gate II directory (usually "C:\Program Files\Black Isle\BGII - SoA" or "C:\Program Files (x86)\Black Isle\BGII - SoA")and run "setup-IlmatarPack.exe".

To uninstall, run "setup-IlmatarPack.exe" again and select the uninstall option.

Compatibility and Installation Order

We have done our best to test on every platform, and this mod should be compatible with all other mods (excluding those which change portraits obviously - for example wonnimchunha's Portrait Mod). Please report any suspected conflicts and we will do our best to make everything work together smoothly. If you do encounter an error, please post on the forums.

Although it is not required for Ilmatar Portrait Pack to function properly, it is always a good idea install the latest version of the BG2 Fixpack.
